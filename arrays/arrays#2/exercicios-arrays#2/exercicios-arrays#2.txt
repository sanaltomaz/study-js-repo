    Programação é prática! 
Dominar o uso de arrays é fundamental em programação e uma base essencial na 
manipulação de dados. 
Praticar regularmente é a chave para se tornar mais confiante!

Nota: Para esses exercício usar necessario estudar arquivos:
    arrays#2.js
    arrays-matrizes.js

* Crie uma função que receba dois arrays e os concatene em um único array.

* Crie um array chamado numeros contendo números de 1 a 10. Utilize o método 
    slice para criar um novo array chamado parteNumeros que contenha apenas 
    os números de índice 3 a 7 de numeros.

* Dado o array frutas contendo frutas que desejamos comprar na feira:
    const frutas = ['Maçã', 'Banana', 'Laranja', 'Limão', 'Abacaxi']
    Utilize o método splice para remover as frutas no índice 2 e 3 e, 
    em seguida, adicione as frutas 'Kiwi' e 'Pêssego' nesses mesmos índices.

* Crie dois arrays chamados menuPrincipal e menuDeSobremesas contendo 
    opções do cardápio de um restaurante. Utilize o método concat para 
    criar um novo array menuCompleto contendo todos os elementos de 
    menuPrincipal seguidos pelos elementos de menuDeSobremesas.

* Crie uma lista bidimensional com 3 linhas e 3 colunas, onde cada 
    elemento seja uma matriz 3x3 com valores iniciando em 1 e 
    aumentando em 1 a cada elemento.
    Dicas:
    comece com um array vazio, por exemplo const matriz = [] e adicione 
    valores nele com push;  
    você pode resolver usando um for dentro de outro for.

* Acesse e imprima o elemento na segunda linha e terceira coluna da lista 
    bidimensional matriz criada no exercício anterior.

* Adicione um novo elemento (por exemplo,15) na terceira linha e segunda 
    coluna da lista bidimensional matriz criada anteriormente.
